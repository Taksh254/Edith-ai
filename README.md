# EDITH AI

EDITH (Even Dead I'm The Hero) is a personal AI assistant project.

## Structure

- `main.py`: Entry point.
- `brain.py`: AI logic (Ollama integration).
- `voice.py`: Speech handling.
- `edith_commands.py`: System commands.
- `memory.py`: Conversation history.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run EDITH:
   ```bash
   python main.py
   ```
